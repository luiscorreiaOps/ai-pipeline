# Instalacao

O ai-pipeline-tool pode ser instalado de duas formas principais: via gerenciador de pacotes pip ou via codigo fonte para desenvolvimento.

## Requisitos

- Python 3.11 ou superior.
- Pip instalado.
- Ambiente Unix-like (Linux ou macOS ).

## Via PyPI (Recomendado)

Para instalar a versao estavel mais recente:

```bash
pip install ai-pipeline-tool
```

## Via Codigo Fonte

Para realizar modificacoes ou utilizar a versao de desenvolvimento:

1. Clone o repositorio:
```bash
git clone https://github.com/luiscorreiaOps/ai-pipeline.git
```

2. Acesse o diretorio:
```bash
cd ai-pipeline
```

3. Instale em modo editavel:
```bash
pip install -e .
```

## Verificacao

Apos a instalacao, valide o funcionamento com o comando:

```bash
ai-pipeline-tool version
```
